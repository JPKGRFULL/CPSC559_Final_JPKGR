# CPSC559_Final_JPKGR
This is the CPSC559 final for John Paul Gomez-Reed, it is based on the linked Midterm Project and the Asteroids Project linked below:

Midterm: https://github.com/Shantanumtk/CPSC-559-CryptoZombies-Dapp/tree/main

Asteroids: https://github.com/MattMuller88/Asteroids-Game-JavaScript

This project is a modification to both projects. To run the project, you can follow the instructions found in the Midterm to create and run the Dapp.

To run the game, you must create a zombie NFT in the Dapp and then either press the button on the Zombie card or when selecting the zombie, press the button in the popup menu.

The game runs the same as the original Asteroids project with some adjustments. The controls from the project are all the same, up arrow key to move forwars, left and right arrow keys to turn, and space bar to shoot your weapon. The difference now is that there are now NFT specific and RPG mechanics. Zombie DNA now changes the color of the ship, the size of the ship. Ships now have two possible weapons, one mapped to space bar, the other mapped to the down arrow key, these weapons are also determined by the Zombie DNA. The Zombie level now also determines the speed of the ship, the range of the weapons, and the difficulty level; the higher the level, the higher these stats. There is also a reset level on the equal key. A reward is dispensed after meeting a threshold for the game as well. If a player exceeds the score of 5000 points, they are granted a reward in zombie coins, their new high score is then added to the minimum score requirement to earn more rewards.
